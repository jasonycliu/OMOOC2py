# OMOOC.py 周任务代码试作

## 4w
- 使用说明
    + 下载4wex0目录下的文件
      * client.py: commond line 客户端
      * mydiary.log: 日记文件
      * bottle.pyc: bottle框架
      * main.py: 服务端
      * mydiary.tpl: bottle框架中用到的模板文件
    
     + 运行服务端: python main.py
     + 用浏览器访问:http://localhost:8080, 就可以在界面上记录日记和查看历史日记
     + 用终端工具 iterm2 运行客户端:python client.py ,根据提示进行记录\查看\日记等操作.
